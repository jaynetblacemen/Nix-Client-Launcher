Nix-client launcher is a Open-source launcher for Nix client, It is made with Golang and GUI is degined using QT framework. The launcher is degine to be extremely lightweight and minimal resource usage. It is native to Linux and work on every distro.

## Features of our launcher

- **Modern UI**: Built with Qt6 for a modern degine and sink with your system theme.
- **Performance First**: Lightweight launcher so your system focus on the game.
- **Modded ready**: Support for modded environments (Fabric) and custom game directories.
- **Auto setup**: It automatically install The best JVM and Nix client resources for you.
- **Resource Management**: Customize RAM allocation per profile to ensure smooth gameplay.
- **Wayland native**: Our launcher is fullly wayland native.

## Usage

Nix-clinet is distributed as an **AppImage**.
